Характеристики:

  • Компактность: занимает менее 100K на диске (когда собран в режиме "lean");
  • Портируемость: собирается почти на любой POSIX-совместимой OS, без налагаемых аппаратных ограничений;
  • Экономичность: работает на устройствах с ограниченными ресурсами (оперативная память, процессор);
  • UDP-to-TCP: мультикаст поток по UDP на входе, юникаст поток по TCP на выходе;
  • RTP трансляция: RTP поверх MPEG-TS на входе, "чистый" MPEG-TS на выходе;
  • Встроенная видеозапись: запись исходящего потока на диск по графику (посредством udpxrec);
  • Встроенный веб интерфейс: управление, вывод статуса и статистики потока через веб интерфейс;
  • Настраиваемость: размер очереди сообщений и буферов настраиваются через опции командной строки.
