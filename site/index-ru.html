Обзор:

udpxy - серверное приложение (daemon) для передачи данных из сетевого потока мультикаст канала (вещаемого по UDP) в HTTP соединение запрашивающего клиента.

udpxy - приложение с открытым кодом, распространяемое бесплатно в соответствие с лицензией GNU GPLv3. Данная статья (на английском языке) доступно разъясняет условия лицензии.

udpxy был написан для последующей сборки и эксплуатации на POSIX-совместимых системах, при минимуме внешних зависимостей и шагов конфигурации; приложение проектировалось с учётом экономного расхода памяти, на устройствах с ограниченными ресурсами (процессор, оперативная память).
