Getting support, requesting features and reporting bugs:

At this point, the best way to ask a question, submit a request or a bug is by creating a topic at the udpxy forum.

Another way to go is through Source Forge, you can report bugs and submit feature requests there too. Source Forge will notify us and provide tracking of your requests.

udpxy binary, when run without parameters outputs an email that one can still use but we would like to kindly ask everyone requiring help to consider posting to the forum as the preferred method. We try to answer everybody and do it as soon as we can.
